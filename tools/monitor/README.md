监控系统
========

## 数据指标

- 访问

    - PV/UV
    - 页面来源
    - 操作系统
    - 浏览器
    - 分辨率
    - 登录率
    - 地域分布
    - 网络类型
    - 访问时段
    - 停留时长
    - 到达深度

- 点击

    - 页面总点击量
    - 人均点击量
    - 流出 url
    - 点击时间
    - 首次点击时间
    - 点击热力图

- 性能

    - 白屏时间
    - 首屏时间
    - 用户可操作时间
    - 页面总下载时间

- 异常

    - 异常的提示信息
    - JS 文件名
    - 异常所在行
    - 发生异常的浏览器
    - 堆栈信息
