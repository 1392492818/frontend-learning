## `startTabBasedApp(params)`
- 打开动画：无，淡入淡出，下滑
- 底栏页签

    - 标签
    - 图标
    - 对应视图
    - 对应视图导航栏标题
    - 对应视图导航条样式
    - 对应视图导航条按钮

- 底栏页签样式

    - 按钮颜色
    - 按钮选中颜色
    - 背景色
